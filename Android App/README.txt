------------------------------------
**       Study Buddy Android      **
------------------------------------

This folder contains the Android 
application of Study Buddy. It is an
Android Studio module, inside of an 
Android Studio project. In order to
run or develop Study Buddy, import the
Android Studio project.

To run the application, enable "Developer
Options" on your Android device, plug it
in to your PC, and click on the green arrow
at the top of Android Studio (make sure
the combo box next to the arrow says
"studybuddy").
